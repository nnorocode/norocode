# Templates sociais

Arquivos editáveis em SVG:

- `post-instagram.svg`: publicação quadrada `1080 × 1080`.
- `story-instagram.svg`: story vertical `1080 × 1920`.
- `capa-linkedin.svg`: capa horizontal `1584 × 396`.

Os arquivos podem ser abertos no Canva, Figma, Inkscape ou Illustrator. Substitua os textos de exemplo sem alterar a paleta principal. Para publicações de produtos específicos, aplique a identidade do produto e mantenha uma assinatura discreta `por NORO TECH`.
